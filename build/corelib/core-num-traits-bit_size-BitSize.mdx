title: "core::num::traits::bit_size::BitSize"

A trait used to retrieve the size of a type in bits.



## Signature

```rust
pub trait BitSize
```



## Trait functions

### bits



Returns the bit size of `T` as a `usize`.

#### Signature

```rust
fn bits() -> u32
```




#### Examples

```rust
use core::num::traits::BitSize;

let bits = BitSize::::bits();
assert!(bits == 8);
```
