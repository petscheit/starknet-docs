title: "core::fmt::into_felt252_based"

Implementations for `Debug` and `LowerHex` for types that can be converted into `felt252` using
the `Into` trait.
## Examples

```rust
impl MyTypeDebug = crate::fmt::into_felt252_based::DebugImpl;
impl MyTypeLowerHex = crate::fmt::into_felt252_based::LowerHexImpl;
```
