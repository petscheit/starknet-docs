## Signature

```rust
pub trait ArrayTrait
```



## Trait functions

### new



Constructs a new, empty `Array`.

#### Signature

```rust
fn new() -> Array
```



#### Examples

```rust
let arr: Array = ArrayTrait::new();

let arr = ArrayTrait::::new();
```

It is also possible to use the `array!` macro to create a new array.
```rust
let arr: Array = array![];
```





### append



Adds a value of type `T` to the end of the array.

#### Signature

```rust
fn append(ref self: Array, value: T)
```



#### Examples

```rust
let mut arr: Array = array![1, 2];
arr.append(3);
assert!(arr == array![1, 2, 3]);
```





### append_span



Adds a span to the end of the array.

#### Signature

```rust
fn append_span, +Drop>(ref self: Array, span: Span)
```



#### Examples

```rust
let mut arr: Array = array![];
arr.append_span(array![1, 2, 3].span());
assert!(arr == array![1, 2, 3]);
```





### pop_front



Pops a value from the front of the array.
Returns `Some(value)` if the array is not empty, `None` otherwise.

#### Signature

```rust
fn pop_front(ref self: Array) -> Option
```



#### Examples

```rust
let mut arr = array![2, 3, 4];
assert!(arr.pop_front() == Some(2));
assert!(arr.pop_front() == Some(3));
assert!(arr.pop_front() == Some(4));
assert!(arr.pop_front().is_none());
```





### pop_front_consume



Pops a value from the front of the array.
Returns an option containing the remaining array and the value removed if the array is
not empty, otherwise `None` and drops the array.

#### Signature

```rust
fn pop_front_consume(self: Array) -> Option, T)>
```



#### Examples

```rust
let arr = array![2, 3, 4];
assert!(arr.pop_front_consume() == Some((array![3, 4], 2)));

let arr: Array = array![];
assert!(arr.pop_front_consume().is_none());
```





### get



Returns an option containing a box of a snapshot of the element at the given 'index'
if the array contains this index, 'None' otherwise.
Element at index 0 is the front of the array.

#### Signature

```rust
fn get(self: @Array, index: u32) -> Option>
```



#### Examples

```rust
let arr = array![2, 3, 4];
assert!(arr.get(1).unwrap().unbox() == @3);
```





### at



Returns a snapshot of the element at the given index.
Element at index 0 is the front of the array.
#### Panics

Panics if the index is out of bounds.
#### Examples

```rust
let mut arr: Array = array![3,4,5,6];
assert!(arr.at(1) == @4);
```





#### Signature

```rust
fn at(self: @Array, index: u32) -> @T
```


### len

#### Signature

```rust
fn len(self: @Array) -> u32
```



#### Examples

```rust
let arr = array![2, 3, 4];
```

### is_empty



Returns whether the array is empty or not.

#### Signature

```rust
fn is_empty(self: @Array) -> bool
```



#### Examples

```rust
let mut arr = array![];
assert!(arr.is_empty());
arr.append(1);
assert!(!arr.is_empty());
```
