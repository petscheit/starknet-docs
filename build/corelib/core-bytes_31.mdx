Definitions and utilities for the `bytes31` type.
The `bytes31` type is a compact, indexable 31-byte type.
## Examples

Creating a `bytes31` from a `felt252`:
```rust
let value: bytes31 = 0xaabb.try_into().unwrap();
```

Accessing a byte by index:
```rust
assert!(value[0] == 0xbb);
```
