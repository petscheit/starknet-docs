Core traits for various operations.
This module provides a collection of essential traits that define common behavior patterns
for Cairo types.
# Main Categories
## Memory Management

- [`Copy`](./core-traits-Copy): Enables value semantics for types
- [`Drop`](./core-traits-Drop): Allows values to be safely discarded
- [`Destruct`](./core-traits-Destruct): Provides custom cleanup behavior for non-droppable types
- [`PanicDestruct`](./core-traits-PanicDestruct): Handles destruction during panic scenarios
## Arithmetic Operations

- [`Add`](./core-traits-Add), [`Sub`](./core-traits-Sub), [`Mul`](./core-traits-Mul), [`Div`](./core-traits-Div), [`Rem`](./core-traits-Rem): Standard arithmetic operators (`+`, `-`, `*`,
`/`, `%`)
- [`DivRem`](./core-traits-DivRem): Combined division and remainder operation
- [`Neg`](./core-traits-Neg): Unary negation (`-`)
## Bitwise Operations

- [`BitAnd`](./core-traits-BitAnd), [`BitOr`](./core-traits-BitOr), [`BitXor`](./core-traits-BitXor): Binary bitwise operations (`&`, `|`, `^`)
- [`BitNot`](./core-traits-BitNot): Unary bitwise complement (`~`)
## Comparison

- [`PartialEq`](./core-traits-PartialEq): Equality comparison (`==`, `!=`)
- [`PartialOrd`](./core-traits-PartialOrd): Ordering comparison (``, `>=`)
## Type Conversion

- [`Into`](./core-traits-Into): Infallible type conversion
- [`TryInto`](./core-traits-TryInto): Fallible type conversion
## Utility Traits

- [`Default`](./core-traits-Default): Creation of default values
- [`Felt252DictValue`](./core-traits-Felt252DictValue): Support for dictionary value types
