The 256-bit unsigned integer type.
The `u256` type is composed of two 128-bit parts: the low part [0, 128) and the high part [128,
256).



## Signature

```rust

#[derive(Copy, Drop, Hash, PartialEq, Serde)]
pub struct u256 {
    pub low: u128,
    pub high: u128,
}
```
