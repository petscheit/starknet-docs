
## [Extern types](./core-gas-extern_types)

| | |
|:---|:---|
| [BuiltinCosts](./core-gas-BuiltinCosts) | Type representing the table of the costs of the different builtin usages. |
| [GasBuiltin](./core-gas-GasBuiltin) | The gas builtin. This type is used to handle gas in the Cairo code. Contains the amount of gas available for the current run. |
| [GasReserve](./core-gas-GasReserve) | Represents a gas reserve. Gas reserves can be created at any point using gas from the gas counter, and can be utilized at a later point in time. |
