Integer types and operations.
This module provides the built-in integer types and their associated operations.
# Integer Types

The following integer types are available:
- Unsigned integers: [`u8`](./core-integer-u8), [`u16`](./core-integer-u16), [`u32`](./core-integer-u32), [`u64`](./core-integer-u64), [`u128`](./core-integer-u128), [`u256`](./core-integer-u256)
- Signed integers: [`i8`](./core-integer-i8), [`i16`](./core-integer-i16), [`i32`](./core-integer-i32), [`i64`](./core-integer-i64), [`i128`](./core-integer-i128)
# Operations

Integer types implement various traits that enable common operations:
- Basic arithmetic via [`Add`](./core-traits-Add), [`Sub`](./core-traits-Sub), [`Mul`](./core-traits-Mul), [`Div`](./core-traits-Div), [`Rem`](./core-traits-Rem) and [`DivRem`](./core-traits-DivRem)
- Bitwise operations via [`BitAnd`](./core-traits-BitAnd), [`BitOr`](./core-traits-BitOr), [`BitXor`](./core-traits-BitXor), and [`BitNot`](./core-traits-BitNot)
- Comparison via [`PartialEq`](./core-traits-PartialEq) and [`PartialOrd`](./core-traits-PartialOrd)
- Safe arithmetic via [`CheckedAdd`](./core-num-traits-ops-checked-CheckedAdd), [`CheckedSub`](./core-num-traits-ops-checked-CheckedSub), [`CheckedMul`](./core-num-traits-ops-checked-CheckedMul)
- Wrapping arithmetic via [`WrappingAdd`](./core-num-traits-ops-wrapping-WrappingAdd), [`WrappingSub`](./core-num-traits-ops-wrapping-WrappingSub), [`WrappingMul`](./core-num-traits-ops-wrapping-WrappingMul)
- Overflow handling via [`OverflowingAdd`](./core-num-traits-ops-overflowing-OverflowingAdd), [`OverflowingSub`](./core-num-traits-ops-overflowing-OverflowingSub), [`OverflowingMul`](./core-num-traits-ops-overflowing-OverflowingMul)
