Set the transaction chain id.

## Signature

```rust
pub fn set_chain_id(chain_id: felt252)
```




#### Arguments

`chain_id` - The chain id to set.
After a call to `set_chain_id`, `starknet::get_execution_info().tx_info.chain_id` will return
the set value.
