Ethereum address type for working with Ethereum primitives.
This module provides the [`EthAddress`](./core-starknet-eth_address-EthAddress) type, which is used when interacting with Ethereum
primitives, such as signatures and L1  L2 messages.



## [Structs](./core-starknet-eth_address-structs)

| | |
|:---|:---|
| [EthAddress](./core-starknet-eth_address-EthAddress) | An Ethereum address, 20 bytes in length. |
